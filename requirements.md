# Code Analyzer Agent

Let's build a Code Analyzer Agent

This agent should follow the below principles:

1. Chain-of-Thought, few-shot, self-consistency
2. Code-focused prompting (generationg, review, debug)
3. Multimodal: images, PDFs, screenshots
4. Build personal prompt library
